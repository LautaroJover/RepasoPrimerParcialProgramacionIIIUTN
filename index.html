10. Conexiones entre temas

Una parte importante de estudiar bien no es memorizar cada apunte por separado, sino ver cómo se conectan.

Componentes del Desarrollo Web

  • La Web: Es el contexto general donde viven páginas y aplicaciones.
  • HTML: Estructura el contenido que el navegador recibe.
  • CSS: Controla la presentación visual de esa estructura.
  • Bloque, línea y modelo de caja: Conceptos que explican cómo se dibujan y distribuyen esos elementos.
  • JavaScript: Agrega lógica y comportamiento a la interfaz.
  • Métodos de arreglos: Herramientas concretas para trabajar con colecciones de datos dentro de JavaScript.
  • Git y ramas: No forman parte de la página en sí, pero son indispensables para organizar el desarrollo del proyecto.

Mirada global

Git organiza el trabajo del programador; la Web define el entorno de ejecución; HTML arma la estructura; CSS define la forma visible; JavaScript define la lógica; y los métodos de arreglos ayudan a procesar datos dentro de esa lógica.
