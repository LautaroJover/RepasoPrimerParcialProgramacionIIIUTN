# Conexiones entre Temas: El Ecosistema del Desarrollo Web

Una parte importante de estudiar bien no es memorizar cada apunte por separado, sino comprender cómo se conectan todos los componentes de un proyecto. Este documento ofrece una visión integrada de las tecnologías fundamentales del desarrollo web y su flujo de trabajo.

---

## 🌐 1. La Web
La **Web** es el contexto general donde viven las páginas y las aplicaciones. Es el entorno global de ejecución donde todo lo demás cobra sentido y se comunica a través de internet.

## 🏗️ 2. HTML (HyperText Markup Language)
**HTML** se encarga de estructurar el contenido que el navegador recibe. Define los cimientos y el esqueleto de cualquier sitio web (párrafos, títulos, imágenes, contenedores).

## 🎨 3. CSS (Cascading Style Sheets)
**CSS** controla la presentación visual de la estructura creada en HTML. 
* **El Modelo de Caja, Bloque y Línea:** Son los conceptos clave que explican cómo el navegador dibuja, dimensiona y distribuye cada uno de los elementos en la pantalla.

## 🧠 4. JavaScript
**JavaScript** agrega la lógica, la interactividad y el comportamiento dinámico a la interfaz de usuario. Permite que la página responda a las acciones del usuario.
* **Métodos de Arreglos (Array Methods):** Son herramientas concretas dentro de JavaScript diseñadas para trabajar, filtrar y transformar colecciones de datos de manera eficiente.

## 🔀 5. Git y Control de Versiones (Ramas)
**Git y sus ramas** no forman parte del código visible de la página en sí, pero son herramientas indispensables para organizar, mantener el historial y colaborar de forma segura en el desarrollo del proyecto.

---

## 👁️ Mirada Global: El Flujo Completo

Para entenderlo de forma sencilla y unificada:

* **Git** organiza el trabajo del programador.
* **La Web** define el entorno de ejecución.
* **HTML** arma la estructura (el esqueleto).
* **CSS** define la forma visible (el diseño y estilos).
* **JavaScript** define la lógica (el comportamiento).
* **Los métodos de arreglos** ayudan a procesar datos dentro de esa lógica.
